So far includes:
* All 300 Kali Tools via kaltoolin
* ctf-tools
* pwntools
* rsa ctf tool
* shellpop
* isf
* ida pro 7 demo
* pyrebox
* unix privesc check
* Hwacha
* Shellter
* Avet
* Frida
